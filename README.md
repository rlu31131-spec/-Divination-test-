# 🔮 靈魂指引 - 占一下卜（Divination-test）

這是一個提供線上占卜和命理分析的靜態網站專案。我們致力於將古老智慧與現代互動技術相結合，為用戶提供全面、深入的自我成長指引。

---

## 🚀 網站特色與功能

1.  **每日占卜（`tarot-single.html`）**
    * 提供單張塔羅牌抽牌功能。
    * 結果為獨創、深入的運勢解讀，涵蓋情感、事業和綜合能量指數。
    * 著重於**原創內容**和**專業度展示**，符合 Google AdSense 的優質內容要求。

2.  **速度反應小遊戲（`game.html`）**
    * 一個結合趣味與專注度的點擊反應測試小遊戲。
    * 旨在增加網站的互動性和用戶停留時間。

3.  **分享機制（`i-ching.html`）**
    * 鼓勵用戶將網站首頁分享給朋友，以啟動「好運連鎖反應」。

---

## 🛠️ 專案技術棧

本專案是一個輕量級的靜態網站，便於部署與維護。

* **HTML5 & CSS3:** 網站結構與基礎樣式。
* **Bootstrap 5.3:** 用於響應式設計和現代 UI 組件。
* **Vanilla JavaScript:** 用於實現塔羅牌抽牌邏輯和速度反應遊戲的互動功能。
* **託管平台:** 專案部署於 GitHub Pages。
